# RestaurantLandingTemplate

## Descrizione
Template modulare per landing page di ristoranti con sistema di prenotazione integrato.

## Utilizzo
1. Clonare i file HTML, CSS e JS
2. Modificare `script.js` con i propri dati del ristorante
3. Sostituire gli URL delle immagini con risorse proprie

## Personalizzazione
- Modificare variabili CSS in `styles.css`
- Adattare `restaurantData` in `script.js`

## Dipendenze
Nessuna. Vanilla JavaScript, CSS e HTML.
